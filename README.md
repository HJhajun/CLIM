# CLIM
CLI Macro

## 설치
```bash
npm install -g cli-macro
```

---

## 사용법
새 매크로 파일 생성:
```bash
clim new mymacro
```

매크로 실행:
```bash
clim run mymacro
```

매크로 편집:
```bash
clim edit mymacro
```

매크로 파일 삭제:
```bash
clim delete mymacro
```

파일 목록 보기:
```bash
clim list
```

## 명령어 옵션
- `new` 또는 `n`: 새 매크로 파일 생성
- `run` 또는 `r`: 저장된 매크로 실행
- `edit` 또는 `e`: 매크로 파일 편집
- `delete` 또는 `del` 또는 `rm`: 파일 삭제
- `list` 또는 `ls`: 현재 폴더의 파일 목록 보기

## 예시
```bash
npm install -g cli-macro
clim new backup
clim run backup
```
